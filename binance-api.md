# Binance API — описание событий

## Тикер (miniTicker)

Снимок рыночных данных за последние 24 часа.

```json
{
  "symbol":     "MBOXUSDT",    // торговая пара
  "price":      "0.01290000",  // последняя цена сделки
  "open_24h":   "0.01330000",  // цена открытия 24ч назад
  "high_24h":   "0.01330000",  // максимальная цена за 24ч
  "low_24h":    "0.01270000",  // минимальная цена за 24ч
  "vol_24h":    "30049999.90", // объём торгов в базовой валюте за 24ч
  "change_pct": "-3.01%"       // изменение цены в % относительно open_24h
}
```

Монета упала на 3%, торговый диапазон узкий (0.0127–0.0133), объём ~30 млн MBOX.

## Сделка (trade)

Конкретная транзакция в реальном времени.

```json
{
  "symbol": "STOUSDT",       // торговая пара
  "price":  "0.13890000",    // цена исполнения сделки
  "qty":    "120.00000000",  // объём в базовой валюте
  "side":   "SELL",          // сторона сделки (см. таблицу ниже)
  "time":   "11:28:48.603"   // время исполнения
}
```

Кто-то продал 120 STO по цене 0.1389 USDT, итого ~16.67 USDT.

### Расшифровка поля `side`

`side` определяется через `isBuyerMaker` из Binance API:

| `isBuyerMaker` | `side` | Смысл |
|---|---|---|
| `true` | `SELL` | покупатель стоял в стакане, продавец пришёл маркет-ордером |
| `false` | `BUY` | продавец стоял в стакане, покупатель пришёл маркет-ордером |

## Доступные WebSocket стримы

### Сделки и цены
- `trade` — каждая сделка в реальном времени (цена, объём, isBuyerMaker)
- `aggTrade` — агрегированные сделки за одну цену/время
- `bookTicker` — лучший bid/ask в реальном времени

### Свечи
- `kline` — OHLCV свечи (1s, 1m, 3m, 5m, 15m, 30m, 1h, 2h, 4h, 6h, 8h, 12h, 1d, 3d, 1w, 1M)

### Тикеры
- `miniTicker` — цена, объём за 24h (упрощённый)
- `ticker` — полный 24h тикер (цена, объём, high, low, кол-во сделок, % изменение)
- `ticker_window` — тикер за произвольное окно (1h, 4h, 1d)
- `avgPrice` — средневзвешенная цена за 5 минут

### Стакан
- `depth` (diff) — дельта изменений стакана
- `depth` (partial) — снимок топ-5/10/20 уровней стакана

### Индексы (для фьючерсов)
- `markPrice` — марк-прайс и ставка финансирования
- `indexPrice` — индексная цена
- `compositeIndex` — составной индекс

### Пользовательские (требуют listenKey)
- `outboundAccountPosition` — обновление баланса
- `executionReport` — статус ордеров (создан, исполнен, отменён)
- `balanceUpdate` — движение средств (депозит/вывод)
