# Стратегия Grid

**Глобальные параметры стратегии:**
- Торговые пары: BTC/USDT, ETH/USDT, SOL/USDT, BNB/USDT.
- Режим: LONG (покупка при падении, продажа при росте). SHORT не используется.
- Тип сетки: Геометрическая (шаг задается в процентах от цены). Это обеспечивает адаптацию к волатильности.
- Бот должен работать 24/7.

**Алгоритм построения сетки:**
1. Пользователь задает диапазон цен:
   - Lower Bound (нижняя граница).
   - Upper Bound (верхняя граница).
2. Количество уровней сетки (Grids): от 10 до 50.
3. Бот автоматически рассчитывает цены уровней по формуле геометрической прогрессии:
   Price[i] = Lower_Bound * (Ratio)^i,
   где Ratio = (Upper_Bound / Lower_Bound) ^ (1 / Grids).

**Механика торговли:**
- При старте бот размещает лимитные ордера на покупку на ВСЕХ уровнях ниже текущей цены.
- При исполнении ордера на покупку на уровне X, бот немедленно размещает лимитный ордер на продажу на следующем уровне выше (X * Ratio).
- При исполнении ордера на продажу на уровне Y, бот немедленно размещает лимитный ордер на покупку на предыдущем уровне ниже (Y / Ratio).
- Таким образом, сетка всегда полностью заполнена ордерами.

**Управление капиталом:**
- Начальный депозит делится на две части: Quote Currency (USDT) и Base Currency (монета).
- Бот рассчитывает объем каждого ордера так, чтобы равномерно распределить доступный капитал по всем уровням сетки.
- Для геометрической сетки объем ордера в Base Currency должен быть одинаковым на каждом уровне.
- Рекомендуемый объем одной сетки: (Общий депозит в USDT / 2) / (Количество уровней) / (Средняя цена уровня).

**Дополнительные функции и защита:**
- **Стоп-лосс:** Уровень цены ниже Lower Bound, при достижении которого бот отменяет все ордера и продает оставшийся актив по рынку (опционально).
- **Трейлинг-ап (Trailing Up):** Если цена уверенно пробивает Upper Bound, бот может автоматически сдвинуть весь диапазон вверх, сохраняя прибыльные позиции.
- **Ребалансировка:** При сильном отклонении рыночной цены от центра сетки бот может предложить добавить средства или перераспределить ордера.
- **Лимит на проскальзывание:** 0.1% для лимитных ордеров (если цена уходит дальше — ждать возврата или отменить).
